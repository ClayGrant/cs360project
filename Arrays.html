Level 1

An array is a list of objects.

int[] arr = new int[10];

This line of code creates an array of integers with the length of 10

From here we can modify and edit the array how we see fit. For example, say we want to make the second element of the array equal to 7

arr[1] = 7;

Notice how we have to use arr[1] to denote the second element, since arrays start at element 0 161.6.5.61

Practice adding elements to the array

arr[8] = 83;
arr[4] = arr[8] - 34;
arr[7] = 1;
